
Tech Spec: ONO / LNO Performance Dashboard +
Brand Reporting (v1.1)

Purpose
Build a single analytics product that:
• Tracks performance across all owned & operated channels/pages
• Combines performance into an "All Channels" view (one unified graph)
• Allows drill-down into any individual channel/page
• Pulls audience demographics (age, gender, geo, etc.) via authenticated access
• Exports clean, professional brand-facing reports for LNO/ONO deals
• NEW: Tracks campaign-level performance against guaranteed deliverables for ROI reporting

Primary Users
Internal
• ONO Operators / Content Managers (daily monitoring, spotting winners/losers)
• Sales / Partnerships (pitch proof, reporting, renewals)
• Leadership / Strategy (portfolio health + trend monitoring)
External
• Brands / Agencies (read-only via report links or PDFs)

Core Use Cases

Portfolio Monitoring
■ See total views across all channels on one chart
■ Spot spikes/drops and identify which channels drove it

Channel Drilldown
■ Click a channel to see its views, top posts, audience, trends

Brand Reporting
■ Select time range + channels + metrics, then export a polished report
■ Send as PDF or shareable link for brands

Demographics Pull

■ Authenticate accounts to access audience age/gender/geo (when available)
5. Campaign ROI Tracking (NEW)
■ Tag content to specific brand deals/campaigns
■ Track delivered performance vs. guaranteed commitments
■ Generate proof-of-performance reports showing exact ROI

Campaign & ROI Tracking (NEW SECTION)
This section enables brands to see exactly what their investment delivered against guaranteed
commitments - the core proof point for amplification deals.

7.1 Campaign Data Model
Each campaign object must include:
• Campaign ID (unique identifier)
• Brand name + campaign/deal name
• Flight dates (start + end)
• Guaranteed deliverables (views, impressions, engagements)
• Deal value (internal only, not shown to brand viewers)
• Channels assigned to campaign
• Tagged content (specific posts tied to this deal)
• Distribution source: ONO channels vs. Clipper Network vs. Both
7.2 Content Tagging
Operators must be able to:
• Tag individual posts to a campaign (single or bulk)
• Auto-tag by hashtag, keyword, or channel assignment
• View all tagged content for a campaign in one list
• Remove/reassign tags if content was misattributed
• Tag content from both ONO channels AND clipper network sources
7.3 Guarantee Tracking & Pacing
Commitment vs. Delivery Dashboard
• Visual progress bar: "X of Y views delivered" (percentage + absolute)
• Pacing indicator: On track / Ahead / Behind (based on flight timeline)
• Projected final delivery (extrapolated from current pace)
• Days remaining in campaign
Alert System
• Flag campaigns pacing below 90% of target with >25% of flight elapsed
• Daily/weekly digest email option for campaign health

7.4 ROI Metrics (Brand-Facing)
Must include:
• Total views delivered
• Total engagements delivered (likes, comments, shares, saves)
• Effective CPM (if brand opts in to show deal value)
• Effective CPV (cost per view)
• Engagement rate (engagements / views)
• Over-delivery percentage: Guaranteed 50M, delivered 67M = 134% delivery
Optional (v1.1 stretch):
• Benchmark vs. industry average CPM/CPV (requires external data source)
• Estimated earned media value (EMV)
7.5 Clipper Network Visibility
For campaigns using the clipper amplification network:
• Aggregate impressions from clipper network (6B+ monthly)
• Breakdown: ONO-sourced vs. Clipper-sourced performance
• Top contributing clippers (anonymized or named, configurable)
• Note: Clipper data may require separate ingestion pipeline (see Section 5)
7.6 Campaign Report Template (Brand-Facing)
Extend the existing report builder (Section 10) to include:
Page: Campaign Performance Summary
• Campaign name + flight dates
• Guaranteed vs. Delivered (with percentage)
• Views, engagements, posts published
• Effective CPM/CPV (optional, brand toggle)
• Visual: delivery progress bar
Page: Content Performance Breakdown
• All tagged posts with individual metrics
• Top 5 posts by views
• Top 5 posts by engagement rate
• Platform breakdown (what % came from TikTok vs. IG vs. YT, etc.)
Page: Distribution Source Breakdown
• ONO channel contribution (views, %)

• Clipper network contribution (views, %)
• Combined reach visualization
7.7 Campaign Lifecycle States
• Draft: Campaign created, not yet active
• Active: Within flight dates, tracking live
• Completed: Flight ended, final metrics locked
• Overdelivering: Active + already exceeded guarantee
• At Risk: Active + pacing below target

Dashboard Requirements
Portfolio View (All Channels)
Must include:
• Combined views over time (single chart)
• Combined watch time (if available), engagement (likes/comments/shares), posts published
• Toggle: daily/weekly/monthly
• Filters: platform, channel group, date range
• Breakdown panel: "Top contributing channels" for the selected range
• NEW: Filter by campaign (show only campaign-tagged content)
Channel View (Per Channel Drilldown)
Must include:
• Views over time
• Top posts (ranked) + per-post metrics
• Audience demographics (age/gender/geo)
• Posting frequency + recent performance summary
• Benchmarked vs portfolio average (optional v1, easy win)
• NEW: Campaign tags on posts (visual indicator)

Metrics (v1.1 Minimum)
Performance
• Views
• Engagement: likes, comments, shares (where available)
• Posts published
• Follower growth (where available)
Audience
• Age distribution
• Gender distribution
• Top countries / cities (where available)
• Language (where available)
Content Ranking
• Top posts by views

• Top posts by engagement rate
• Best day/time performance (optional)
Campaign ROI (NEW)
• Guaranteed vs. delivered (views, engagements)
• Delivery percentage
• Effective CPM / CPV
• Pacing status
10) Brand Report Builder + Export Requirements (Critical)
10.1 Report Types
A) Deal Recap Report (default)
• Time range + selected channels
• Portfolio totals + key charts
• Top channels + top posts
• Audience demographics summary
• Key takeaways section (editable text)
• NEW: Campaign performance vs. guarantee section
B) Campaign ROI Report (NEW)
• Campaign-specific (vs. portfolio-wide)
• Guaranteed vs. delivered with visual progress
• Tagged content breakdown
• Distribution source breakdown (ONO vs. Clipper)
• Effective CPM/CPV (optional toggle)
• "What your investment delivered" executive summary
C) Media Kit Snapshot (optional v1.1)
• Last 30/60/90 days
• Portfolio size + demographic highlights
• Top content examples
• "What brands get" positioning

10.2 Report Builder UI
User selects:
• Date range
• Channels included (or "All ONO/LNO")
• NEW: Campaign (or "No campaign filter")
• Platforms included
• Metrics included
• Brand name + campaign name (for header)
• Optional notes/summary bullets (editable)
• NEW: Toggle: Show CPM/CPV (yes/no)
• NEW: Toggle: Show guarantee vs. delivered (yes/no)
Output preview
• Live preview of report layout before exporting
• Choose layout style: Clean PDF (1-5 pages) or Deck-style PDF (slide-like pages)
10.3 Export Formats
Must support:
• PDF export (primary)
• CSV export (raw data)
• Shareable link (read-only) with: password optional, expiration date optional, watermark optional
10.4 Report Design Requirements ("Premium / Sexy")
Brand-facing styling principles:
• Minimal, clean typography
• Whitespace-heavy layout
• Simple charts (no clutter)
• Consistent spacing + alignment
• Muted, premium color palette (black/white/gray + one accent)
• Your logo + report title on cover
• Professional footer: date generated + time range + confidentiality note (optional)
10.5 Report Content Template (Default Pages)
Page 1 - Cover: Brand Name, Campaign/Deal Name, Time range, Title, Logo

Page 2 - Executive Summary: Total views, engagements, # posts, Top 3 channels, bullet insights.
NEW: Guaranteed vs. Delivered summary (if campaign selected)
Page 3 - Performance Chart: Combined view graph over time, Channel contribution breakdown
Page 4 - Audience: Age distribution, Gender, Top geos
Page 5 - Top Content: Top posts list with thumbnails (if accessible), Views + engagement per post
Page 6 - Campaign ROI (NEW): Delivery progress bar, Guaranteed vs. Actual, CPM/CPV (optional),
Distribution breakdown
11) Roles & Permissions
• Admin: connect accounts, manage org, create campaigns, export + share
• Internal: view dashboards, tag content to campaigns, generate reports
• Brand Viewer: view report links only (no raw dashboard access, no campaign editing)
12) Data Refresh & Reliability
• Daily automated refresh (minimum)
• Manual refresh button with rate-limit handling
• Backfill last 90 days on first connect (or more if API allows)
• Clear indicators for missing data (e.g., "Demographics unavailable for this platform/account")
• NEW: Campaign metrics refresh hourly during active flights
13) Non-Functional Requirements
Performance
• Portfolio view loads in <2 seconds for typical org size
• Report export completes in <30 seconds (typical)
• Campaign dashboard loads in <3 seconds
Security
• Encrypted tokens
• Audit log: account connected/disconnected, report generated, link shared, campaign
created/modified
• Share links are read-only
• Row-level access control per org
• Deal values (pricing) never exposed to Brand Viewer role
Usability

• Extremely simple navigation
• 2-click path to: "Export brand report"
• 2-click path to: "View campaign ROI"
• Mobile-friendly report viewing (share link)
14) Success Criteria
Internal
• No more manual compiling for ONO/LNO reporting
• Daily monitoring becomes a single dashboard workflow
• Operators can isolate which channels drive performance spikes quickly
• NEW: Campaign pacing visible at a glance; at-risk campaigns flagged automatically
External
• Reports look client-ready with minimal editing
• Brands can understand: reach, audience, and proof of performance in <2 minutes
• NEW: Brands can see exactly what their investment delivered vs. what was guaranteed
• NEW: ROI metrics (CPM/CPV) demonstrate value vs. industry alternatives
